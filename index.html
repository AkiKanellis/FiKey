FiKey

FiKey is a website registration implementation based on the U2F second-factor authentication from FIDO Alliance.

Getting started

1. Installing

Make sure you have the latest Java SDK installed.

Make sure you have the latest Maven

2. Compiling

Enter the server module:

cd server

Package with Maven:

mvn package

3. Starting

Make sure your directory contains these files:

  • fikey-x.x.x.jar
  • fikey-config.yml
  • keystore.jks

Start the server running the command:

java -jar fikey-x.x.x.jar server fikey-config.yml

Enter the website by pointing a U2F compatible browser to:

https://localhost:8080/

Stop the server on the terminal:

Ctrl + C

Third-party libraries

  • Dropwizard 0.8.1
  • Yubico U2F
